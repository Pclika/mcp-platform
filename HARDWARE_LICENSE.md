# Hardware License

Hardware design files in this repository, when present and unless otherwise noted, are intended to be licensed under:

- `CERN-OHL-W-2.0`

Official source:

- https://cern-ohl.web.cern.ch/

## Scope

This applies to hardware design artifacts such as:

- schematics
- PCB design files
- hardware CAD files
- manufacturing outputs intentionally released with the project

## Notes

- This repository does not yet include the full first-party hardware design file set.
- When hardware design files are added, maintainers should include a copy of the official `CERN-OHL-W-2.0` text in a dedicated license file if needed for downstream packaging workflows.
- Brand names and logos are not granted by this hardware license.
