# Licensing

This repository uses a split licensing model because it is intended to cover software, hardware design work, documentation, and brand assets.

## Code

Unless otherwise noted, source code and software in this repository are licensed under:

- `Apache-2.0`

See:

- [../../LICENSE](../../LICENSE)

## Hardware Design Files

When hardware design files are added to this repository, they should be licensed under:

- `CERN-OHL-W-2.0`

See:

- [../../HARDWARE_LICENSE.md](../../HARDWARE_LICENSE.md)

## Documentation

Unless otherwise noted, documentation content is licensed under:

- `CC BY 4.0`

See:

- [../../DOCS_LICENSE.md](../../DOCS_LICENSE.md)

## Brand Assets

The Pclika name, logos, and brand materials are not automatically covered by the open-source licenses above.

See:

- [../../TRADEMARKS.md](../../TRADEMARKS.md)

## Notes

- Third-party dependencies may carry their own licenses.
- Future hardware CAD, PCB, and mechanical assets should include file-level or folder-level notices where useful.
- If a file or folder is under a different license, it should say so clearly.
