# 发货履行 SOP

**Pclika MCP Platform — 订单履行标准操作程序 v0.1**

---

## 订单流程总图

```
客户下单（网站 / 邮件 / 淘宝）
  └─ 订单登记（ops/orders/YYYYMMDD-NNN.md）
       └─ 备料核实（库存充足 / 缺货提醒）
            └─ 套件组装
                 └─ 出厂测试（通过后贴合格贴纸）
                      └─ 打包装箱
                           └─ 快递取件 / 寄出
                                └─ 物流号通知客户
                                     └─ 收货确认（7 天后跟进）
```

---

## SOP-F01：订单登记

**触发：** 收到下单通知（邮件 / 平台 / 微信）

1. 在 `ops/orders/` 创建订单记录文件，命名：`YYYYMMDD-NNN.md`

```markdown
# 订单 YYYYMMDD-NNN

**客户：** 姓名
**邮箱：** example@email.com
**电话：** 1xx-xxxx-xxxx
**地址：** 省市区街道，邮编
**订购产品：**
  - KIT-MCP-01 × 1
  - MOD-SEN-01 × 2
**金额：** ¥457.00
**支付方式：** 支付宝 / 微信 / 转账
**下单时间：** YYYY-MM-DD HH:MM
**预计发货：** YYYY-MM-DD（下单后 3 工作日）
**状态：** 待处理
```

2. 核对付款是否到账（付款到账后才进入备货流程）

## SOP-F02：备料与套件组装

### 库存核查

| 物料 | 安全库存 | 补货周期 |
|------|---------|---------|
| ESP32-S3 底板 v0.1 | ≥ 20 块 | JLCPCB 2 周 |
| DHT22 温湿度传感器 | ≥ 50 个 | 立创 3–5 天 |
| SSD1306 OLED 0.96" | ≥ 30 个 | 立创 3–5 天 |
| OV2640 摄像头 | ≥ 15 个 | 立创 5–7 天 |
| USB-C 数据线 1m | ≥ 50 根 | 淘宝 3–5 天 |
| 杜邦线套装 | ≥ 100 包 | 淘宝 3–5 天 |
| 快速开始卡片 | ≥ 100 张 | 印刷厂 5 天 |
| 包装盒（各规格）| ≥ 30 套 | 定制厂 7 天 |

**库存预警：** 任一物料低于安全库存 50% 时自动触发补货流程。

### KIT 组装 SOP

**KIT-MCP-01 Basic 标准组装步骤（预计 5 分钟/套）：**

```
□ 取 ESP32-S3 底板 1 块（已测试，已贴合格贴纸）
□ 取 DHT22 × 1（检查外观无损）
□ 取 SSD1306 OLED × 1（检查排针无弯曲）
□ 取杜邦线套装 × 1（10 根 4P 母对母）
□ 取 USB-C 数据线 × 1
□ 取快速开始卡片 × 1（检查正反面印刷清晰）
□ 防静电袋：底板装入防静电袋，密封
□ 内衬 EVA 泡棉：按槽位摆放所有物料
□ 放入包装盒，盒盖扣合
□ 外贴产品标签（SKU / 序列号 / QC 通过）
□ 套上外袋（防水塑料袋 + 快递标签）
```

## SOP-F04：发货操作

### 国内快递

**推荐：顺丰经济快递（2–3 天）或 京东快递（北方地区快）**

| 重量 | 首选快递 | 参考运费 |
|------|---------|---------|
| < 1kg | 顺丰陆运 | ¥8–12 |
| 1–3kg | 顺丰陆运 | ¥12–18 |
| > 3kg（Edu Lab 套装）| 顺丰标快 | ¥20–35 |

**步骤：**

1. 在快递网站/小程序下单寄件
2. 填写收件人信息（来自订单记录）
3. 描述：「电子产品/嵌入式开发套件」，保价金额 = 产品售价
4. 打印或截图运单号，更新订单记录 `状态: 已发货`
5. 在 30 分钟内发邮件/短信通知客户运单号

### 国际发货

**推荐：DHL Express（5–7 天，美/欧）或 小包（15–30 天，成本低）**

| 目的地 | 方式 | 参考时效 | 估算运费 |
|--------|------|---------|---------|
| 美国 | DHL Express | 5–7 天 | $25–45 |
| 欧洲 | DHL Express | 5–7 天 | $25–50 |
| 东南亚 | 小包/EMS | 10–20 天 | $8–15 |

**注意：**
- 申报价值如实填写（不能低报，风险归客户承担）
- 电子产品含锂电：需标注「Contains Lithium Battery」
- 不含锂电的产品（底板 v0.1 无电池）可走普通航空小包

## SOP-F05：售后与退换货

### 退换货政策

| 情况 | 处理方式 | 时限 |
|------|---------|------|
| 硬件质量问题（非人为损坏）| 免费换货 | 收货后 1 年内 |
| 死机 / 固件不可烧录 | 免费换货或修复 | 收货后 1 年内 |
| 人为损坏（短路/过压）| 按成本价提供换板 | — |
| 不喜欢 / 不需要 | 支持 7 天无理由退货（未拆封）| 收货后 7 天 |

### 换货流程

```
1. 客户发邮件描述问题 + 照片/视频
2. 我方 48 小时内确认是否属于质量问题
3. 确认后：
   a. 国内：寄出新品（顺丰到付），客户收到后原件寄回
   b. 国际：视情况远程修复（固件）或本地维修授权
4. 收到退货后记录问题类型，更新质量报告
```
