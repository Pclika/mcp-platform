## 定价公式

```
目标零售价 = (BOM 成本 + 组装成本 + 包装成本 + 物流成本) × 毛利倍率 × 平台费系数
```

| 分项 | Basic Kit | Pro Kit | Industrial |
|------|-----------|---------|------------|
| BOM 成本（小批量 50 件）| ¥115 | ¥310 | ¥780 |
| 组装 + 测试 | ¥30 | ¥45 | ¥80 |
| 包装 + 配件 | ¥15 | ¥20 | ¥25 |
| 物流（国内）| ¥12 | ¥15 | ¥20 |
| **总成本** | **¥172** | **¥390** | **¥905** |
| 毛利倍率 | 2.3× | 3.1× | 3.3× |
| **建议零售价** | **¥399** | **¥1,199** | **¥2,999** |
| **毛利率** | ~57% | ~68% | ~70% |

> BOM 成本基于 LCSC/立创商城 50 件价格估算，量产后（≥500 件）BOM 成本可降 30–40%。

## 特殊定价政策

### 教育折扣

- 认证高校 / 职业学校：套件价格 7 折
- 需提供院校采购函
- Edu Lab 套装（KIT-MCP-04）含教育支持，已体现折扣

### 开发者计划（DevProgram）

- GitHub 贡献者（合并 PR ≥ 2 个）：Basic Kit 免费 1 件
- 开源项目合作：Pro Kit 5 折
- 写测评文章或视频教程：产品换测评（公允价值交换）

### 样品政策

- ToB 客户首次合作：1 件 Basic Kit 免费寄样（快递到付）
- 年采购额 ≥ ¥50,000 客户：免费年度技术支持升级

## 定价调整机制

- 每季度审查一次 BOM 成本
- 元器件涨价 ≥ 15%：提前 30 天公告调价
- 重大版本升级（v0.x → v1.0）：设 60 天过渡期，旧版本限量清货
