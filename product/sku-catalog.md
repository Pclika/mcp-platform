## SKU 编码规范

```
[产品类别]-[系列]-[序号]-[变体]

示例：
KIT-MCP-01        →  KIT产品线，MCP系列，第01号
KIT-A             →  套件A（Starter）
MOD-SEN-01        →  模块，传感器类，第01号
PCB-ESP32S3-01    →  裸板，ESP32-S3底板，第01版
SVC-PROTO-01      →  服务，原型整合，标准方案
```
