# Pclika System Context

Use this repository as the source of truth for the Pclika embedded platform.

Before writing code:

1. Read the root README
2. Read AGENTS.md
3. Read the relevant hardware and software docs
4. Inspect the closest example

When adding a feature:

1. Extend the hardware definition if needed
2. Extend the runtime capability
3. Extend the MCP tool surface if needed
4. Add or update an example
5. Update docs

The first platform target is ESP32-S3.
