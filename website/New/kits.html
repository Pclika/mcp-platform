Build your own — Module Catalog

Every module below is compatible with any Pclika baseboard and exposes its capability through the standard MCP tool API. Mix and match to build your kit.

DHT22
Temperature + humidity. 1-wire. 0.1°C resolution. sensor_read
BME280
Temperature, humidity, barometric pressure. I2C. sensor_read
SHT31
Precision temperature + humidity. ±0.3°C. I2C. sensor_read
SCD40
CO2 + temperature + humidity. Photoacoustic. I2C. sensor_read
BH1750
Ambient light intensity. Lux. I2C. sensor_read
MPU6050
6-axis IMU. Accel + Gyro. I2C. sensor_read
VL53L0X
Time-of-flight distance 0–2m. I2C. sensor_read
SSD1306 OLED
128×64 display. I2C. 4 text lines. display_text
ST7789 TFT
240×240 color display. SPI. display_text + graphics
SG90 Servo
180° micro servo. PWM. servo_move (ch0)
PCA9685
16-channel PWM servo driver. I2C. servo_move (ch0–15)
Relay Module
5V single-channel relay. gpio_write to toggle. 10A/250V
