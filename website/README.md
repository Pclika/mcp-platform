# Website

This directory is reserved for the Pclika front-end experience.

The site should combine:

- product selection
- documentation access
- module capability browsing
- GitHub entry points
- B2B service conversion

The first website milestone is an information architecture and content-first prototype.

## Current Files

- `index.html`: first homepage prototype
- `styles.css`: first visual system and layout styles

## Homepage Focus

The current homepage introduces:

- what Pclika MCP Platform is
- current project status
- product direction
- module families
- open-source structure
- roadmap

## Next Front-End Steps

- turn the static homepage into a reusable component-based front end
- add docs and GitHub entry pages
- add kit and module selection flows
- add B2B and support conversion paths
